# Personal Portfolio Website

This is my personal portfolio website built using **Next.js**, **React**, and **Styled Components**.  
It showcases my projects, skills, and experience in a modern and responsive design.

## 🚀 Live Demo
--- Netlify : https://aayu-portfolio-site.netlify.app/
--- Github: https://github.com/Aayush-Sharm/portfolio

## ✨ Features
- Modern, clean, and responsive design  
- Built with **Next.js (Static Export)**  
- Styled using **Styled Components**  
- Smooth animations and interactions  
- Projects, About, Skills, and Contact sections  
- Easy to customize and expand  

---

## 🛠️ Tech Stack
- **Next.js 10**
- **React 17**
- **Styled Components**
- **React Icons**

---

## 📁 Project Structure
components/ → Reusable UI components
layouts/ → Page layouts
pages/ → Main pages (Home, Projects, About...)
styles/ → Global styles & theme
public/ → Assets and static files
